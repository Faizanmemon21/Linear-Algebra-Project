# 🔐 Hill Cipher Encryption

A Python-based implementation of the **Hill Cipher**, developed as a university project for the **Cryptography** course. This project demonstrates how classical cryptographic techniques use matrix multiplication and modular arithmetic to securely encrypt and decrypt text messages.

---

## Features

- 🔒 Text Encryption
- 🔓 Text Decryption
- 🔢 Matrix-Based Encryption
- ➗ Modular Arithmetic
- 🛡️ Classical Cryptography
- 🐍 Python Implementation

---

## Technologies Used

- Python 3
- Cryptography
- Matrix Operations
- Modular Arithmetic

## How to Run

### Requirements

- Python 3.x

### Steps

1. Clone or download this repository.
2. Open the project folder in your preferred IDE or terminal.
3. Navigate to the project directory.
4. Run the main program:

```bash
python main.py
```

or

```bash
python3 main.py
```
